ABOUT the project

What is SpaceXpanse?


SpaceXpanse is a decentralized open source information registration and value transfer project based on the Namecoin cryptocurrency.

It is primarily aimed at decentralised blockchain worlds creation.

SpaceXpanse ROD Core wallet is the first layer of the SpaceXpanse Platform.
